The Story

About

FINLAC is a local-first budgeting tool built for students who need a simple, private way to stay on top of their spending. It runs entirely in the browser — no accounts, no servers, no data leaving your device.

Students can log transactions across categories like Food, Books, and Transport, set a monthly spending cap, and instantly search or sort their records. The dashboard surfaces totals, top spending categories, and a 7-day trend so you can see where your money is going at a glance. Data can be exported and imported as JSON for backup or transfer between devices.

Features

  • Local-first — all data stays in your browser
  • Live regex search across all fields
  • Sort by date, description, or amount
  • Monthly budget cap with visual progress bar
  • Manual currency conversion (two configurable currencies)
  • 7-day spending trend chart
  • JSON export / import for backup and migration
  • Accessible: keyboard navigation, ARIA live regions, skip link
